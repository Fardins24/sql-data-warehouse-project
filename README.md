# Data Warehouse and Analytics Project
This repository contains an end-to-end data warehouse and analytics project, showcasing how raw data can be transformed into meaningful insights. The project covers the full lifecycle of a modern data warehouse â€” from data modeling and ingestion to transformation and analytics.

The goal of this project is to demonstrate best practices in data engineering and analytics, including structured schema design, layered data architecture (bronze, silver, gold), and SQL-based transformations. It is designed as a portfolio project to highlight practical, industry-relevant skills used in real-world data platforms.

#### Tech Stack

Database: PostgreSQL 18
Database Administration: pgAdmin 4 (Desktop & Browser UI)
Query Language: SQL
Data Modeling: Layered architecture (Bronze, Silver, Gold schemas)
Development Environment: macOS
Version Control: Git & GitHub
